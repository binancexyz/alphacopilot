# Scripts

## `check_all.sh`
Runs local compile and test checks.

## `setup_local_env.sh`
Creates `.venv`, installs requirements, and prepares a local `.env` from `.env.example`.

## `run_and_test.sh`
Activates `.venv`, runs checks/tests, and performs a quick smoke test.

## `fill_env_checklist.sh`
Shows which live env values should be filled before real usage.

## `start_api.sh`
Starts the FastAPI service using `API_HOST` and `API_PORT`.

## `summary.sh`
Prints a quick project summary.

## `install_square_diary_cron.sh`
Installs the current Binance Square cron schedule in `Asia/Phnom_Penh`.
