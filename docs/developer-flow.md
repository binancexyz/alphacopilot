# Developer Flow

## Day-to-day flow
1. Pull latest changes
2. Read `docs/maintainer-summary.md`
3. Run:
   ```bash
   make check
   make test
   ```
4. Work on one command path or one doc area at a time
5. Re-run checks/tests
6. Commit in small, understandable units
