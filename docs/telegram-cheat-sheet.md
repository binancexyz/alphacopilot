# Telegram Cheat Sheet

Use these commands in chat:

- `/brief <symbol>` — quick asset read
- `/brief <symbol> deep` — deeper asset read
- `/signal <symbol>` — setup + invalidation + risk
- `/holdings` — your posture
- `/holdings <address>` — wallet behavior
- `/watchtoday` — market board
- `/audit <symbol>` — safety check

## Best daily flow

1. `/watchtoday`
2. `/holdings`
3. `/brief <top name>`

## Best asset flow

1. `/brief <symbol>`
2. `/signal <symbol>`
3. `/audit <symbol>`

## Best publish flow

1. `/watchtoday`
2. draft post
3. confirm
4. publish
