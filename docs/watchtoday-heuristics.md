# `/watchtoday` Heuristics

## Goal
Filter market noise into a short daily research brief.

## Inputs
Use `WatchTodayContext`:
- top_narratives
- strongest_signals
- risk_zones
- market_takeaway
- major_risks

## Signal Quality Rules
### High
- multiple narratives
- multiple strong signal areas
- broad enough context to prioritize confidently

### Medium
- some useful narratives or signals exist
- but quality still varies

### Low
- weak or incomplete market context

## Key idea
The value is not showing everything.
The value is prioritizing what actually matters today.
