# Project Goals

## Near-term goal
Ship a strong contest-ready research copilot scaffold with clear positioning, command architecture, and live integration plan.

## Mid-term goal
Wire real OpenClaw + Binance Skills Hub runtime flows into the prepared command architecture.

## Long-term goal
Evolve AlphaCopilot into a genuinely useful crypto signal-and-risk copilot with stronger heuristics, better user experience, and reusable runtime-independent logic.
