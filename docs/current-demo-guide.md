# Current Demo Guide

A consolidated current demo reference for Bibipilot.

## Goal
Show Bibipilot as a clean research -> judgment -> publishing product with the current command architecture.

## Best demo order
1. `/brief BNB`
2. `/portfolio`
3. `/watchtoday`
4. `/token BNB`
5. `/signal BNB`
6. `/wallet <address>`
7. Binance Square publishing proof

This order matches the current product story better than older command-first sequences.

---

## Opening line
Bibipilot is a Binance-native copilot that checks what’s moving, filters what’s dangerous, and publishes what’s worth attention.

---

## Demo flow

### 1. `/brief BNB`
**What to show**
- fast default market read
- source/context framing
- confidence-aware compact answer

**What it proves**
- Bibipilot can compress useful judgment quickly
- the product does not rely on walls of text to feel intelligent

### 2. `/portfolio`
**What to show**
- private posture
- freshness / recent change context
- grouped exposure and top holdings

**What it proves**
- Bibipilot is not just public market commentary
- private posture intelligence is a real differentiator

### 3. `/watchtoday`
**What to show**
- market lanes
- ranking/selectivity
- daily board feel

**What it proves**
- Bibipilot can prioritize attention at the market level
- this is a habit-forming front-page command, not just a single-asset tool

### 4. `/token BNB`
**What to show**
- deeper asset judgment
- audit-aware context
- richer risk/why/watch structure

**What it proves**
- Bibipilot can go deeper than `/brief` without becoming noisy

### 5. `/signal BNB`
**What to show**
- timing
- fragility
- explicit invalidation

**What it proves**
- Bibipilot is not a hype repeater
- it can explain what would break the setup

### 6. `/wallet <address>`
**What to show**
- follow verdict
- behavior/profile framing
- concentration/activity interpretation

**What it proves**
- Bibipilot interprets behavior, not just balances

### 7. Binance Square publishing proof
**What to show**
- one recent market-aware post
- one recent reflective/nightly post
- evidence that the product closes the loop into publishing

**What it proves**
- Bibipilot is not only an internal analysis tool
- it has a real output loop

---

## Fallback guidance
- If live evidence is thin, emphasize honesty and trust framing.
- If a command is partial, explain that Bibipilot surfaces uncertainty explicitly instead of hiding it.
- If you need the shortest path, demo `/brief`, `/portfolio`, `/watchtoday`, and publishing proof.

---

## Closing line
Bibipilot turns noisy crypto signals into clearer judgment, then turns that judgment into publishable Binance Square output.
