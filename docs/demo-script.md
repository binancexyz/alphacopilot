# Demo Script

## Opening
Crypto attention moves faster than clean judgment.

**Bibipilot** is a Binance-native copilot that checks what’s moving, filters what’s dangerous, and publishes what’s worth attention.

The product is built around a simple loop:

**research -> judgment -> publishing**

---

## Demo 1 — `/price BNB`
### Show
- compact market card
- fast price/trend/rank view

### Narration
Start with grounding.
This shows Bibipilot can give a clean market truth immediately, not a wall of text.

### Judge takeaway
This feels like a real product surface, not raw terminal output.
