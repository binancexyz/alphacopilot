# Medium Submission

## Project Name
Bibipilot

**Short Name:** Bibipilot

## Tagline
Bibipilot checks what’s moving, filters what’s dangerous, and publishes what’s worth attention.

## Description
Bibipilot is a Binance-native research copilot built for a simple problem:

**crypto attention moves faster than clean judgment.**

Users can see token movement, wallet activity, ranking changes, narrative rotation, and smart-money signals everywhere, but that still does not answer the questions that matter:
- what is real?
- what is risky?
- what is late?
- what is worth publishing?

Bibipilot closes that gap.

It uses Binance Skills as the evidence layer, then adds a judgment layer on top through command-specific outputs such as:
- `/price`
- `/brief`
- `/risk`
- `/audit`
- `/token`
- `/signal`
- `/wallet`
- `/watchtoday`
- `/meme`

Then it closes the loop by publishing directly to Binance Square.

## What makes it different
Most crypto tools surface movement.
Bibipilot tries to answer whether that movement deserves attention, caution, or publication.

Its strength is the full workflow:

**research -> judgment -> publishing**

## Skills used
- `query-token-info`
- `query-token-audit`
- `query-address-info`
- `trading-signal`
- `crypto-market-rank`
- `meme-rush`
- `square-post`

## Why it matters
Bibipilot turns noisy crypto signals into clearer judgment, then turns that judgment into useful Binance Square output.
