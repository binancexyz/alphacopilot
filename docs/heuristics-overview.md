# Heuristics Overview

## Why heuristics exist
AlphaCopilot should not dump raw tool data directly.
Heuristics provide a first-pass interpretation layer.

## Current heuristic areas
- token signal quality
- token conviction
- signal quality from signal context
- wallet signal quality
- market-watch signal quality

## Future refinement
Once real payloads are available, heuristics should be calibrated against actual examples instead of only scaffold assumptions.
