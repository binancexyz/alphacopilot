# MVP Definition

## Product promise
Binance Alpha Copilot helps users understand not just what is moving, but whether it is actually worth attention.

## MVP commands
- `/token <symbol>`
- `/wallet <address>`
- `/watchtoday`
- `/signal <token>`

## MVP output shape
- Quick Verdict
- Signal Quality
- Top Risks
- Why It Matters
- What To Watch Next

## MVP differentiators
- signal + risk together
- wallet behavior interpretation
- risk tags
- conviction label
- concise output format

## MVP non-goals
- full auto-trading
- giant frontend dashboard
- portfolio syncing
- excessive feature surface
