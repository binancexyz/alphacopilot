# Maintainer Checklist

## Before stepping away
- [ ] `make check`
- [ ] `make test`
- [ ] update docs if behavior changed
- [ ] commit small, clear changes
- [ ] push to GitHub

## Before coming back
- [ ] read `docs/maintainer-summary.md`
- [ ] read `docs/live-integration-checklist.md`
- [ ] decide whether you are doing repo polish or live runtime wiring
