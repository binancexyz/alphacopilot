# One-Minute Overview

Bibipilot is a **Binance-native signal-and-risk research copilot** designed for the OpenClaw + Binance Skills Hub ecosystem.

## What it does
- Analyzes tokens, signals, wallets, and market narratives
- Filters noise into conviction-driven research briefs
- Assesses risk, security audits, and smart-money signals
- Publishes insights to Binance Square
- Tracks Binance ecosystem hiring as intelligence context

## What makes it different
It does not just show what is moving. It helps users understand **whether it is actually worth attention** — and then turns that insight into publishable output.

## How it works
1. **Research** — gather evidence via Binance Skills Hub (token info, audit, signals, rankings, meme data)
2. **Judgment** — interpret signal quality, risk, and conviction through Python analyzers
3. **Publishing** — share results to Binance Square with a premium daily posting engine

## Key numbers
- **10 research commands** — price, brief, token, signal, wallet, risk, audit, watchtoday, meme, careers
- **8 API endpoints** — FastAPI REST service (v0.2.1) with security middleware
- **7 diary series** — scheduled premium daily publishing engine
- **6 Binance Skills** — core evidence layer
- **Mock and live modes** — offline development and real integration

## Tagline
**Less noise. Better conviction.**
