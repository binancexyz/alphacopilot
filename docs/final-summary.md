- core commands are `/brief`, `/token`, `/signal`, `/portfolio`, `/wallet`, `/watchtoday`
- specialist commands are `/price`, `/risk`, `/audit`, `/meme`, and `careers`
- `watch today` should remain an alias, not a separately promoted command
